SLMS (Srinivas Library Management System)

Short Project Overview: SLMS is a modern web-based library management system developed for Srinivas University to digitize and simplify the operations of a school library. It allows administrators to efficiently manage books, authors, students, and book transactions, while students can search, view, and track their borrowed books through a secure portal.

Project Details

  • Platform: Web-Based Application (MERN Stack)
  • Frontend: React.js, HTML5, CSS3, JavaScript, Material UI
  • Backend: Node.js, Express.js
  • Database: MongoDB (via Mongoose)
  • APIs: RESTful APIs with secure authentication
  • Libraries/Tools: Axios, Font Awesome, Cloudinary

Key Functionalities

  • Secure login/logout for admins and students
  • Admin Dashboard with total stats and quick actions
  • Category & Author Management (CRUD)
  • Book Management with image upload and details
  • Issue & Return book module with due date tracking
  • Student profile management and viewing borrowed books
  • Real-time search for available books
  • Role-based access control and password update options
  • Responsive design compatible with all major browsers
